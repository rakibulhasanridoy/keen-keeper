#  Keen Keeper

A personal relationship manager that helps you stay in touch with the people you care about.Your personal friend tracker — because great relationships don't maintain themselves.

---

## 🔹 Description

*Keen Keeper is a friendship tracking web app built with React. It lets you log your interactions — calls, texts, and video chats — with friends and visualizes your communication habits over time. The app features a friends directory with status tracking (On Track, Almost Due, Overdue), a live timeline of all logged check-ins, and an analytics page that breaks down your interactions by type. A context-powered state system keeps your timeline and homepage stats in sync in real time, while toast notifications give instant feedback when you log a check-in.

## ✨ Features

### 🔀 Dynamic Navigation System
- Easily navigate between different routes directly from the navbar without reloading the page.

### 👤 Interactive Friend Details
- Clicking on a friend card dynamically updates the URL and displays detailed information about that specific friend.

### 📞💬🎥 Multiple Communication Options
- From the friend details page, users can initiate calls, send messages, or start video interactions.

### 🕒 History Timeline with Local Storage

- All call, text, and video activities are stored in local storage
A dedicated timeline route displays the complete interaction history
Filter functionality allows users to view:
- -  Only Calls
- - Only Texts
- - Only Videos
- Data can be sorted based on time (e.g., most recent or shortest duration)

### 📊 Statistics & Data Visualization
- A separate stats route displays graphical representations of call, text, and video activity.

### ⏳ Loading Fallback Spinner
- A smooth loading spinner is shown while data or routes are being loaded. 

### ❌ Custom 404 Error Page
- A user-friendly custom 404 page is displayed when navigating to an undefined route.

## 🚀 Run Locally

Follow these steps to run the Keen Keeper on your local machine:

## 📌 Prerequisites

Make sure you have the following installed:

- 🟢 Node.js (v14 or higher)
- 📦 npm (comes with Node.js)
- 💻 A code editor like VS Code
- 📥 1. Clone the Repository
```js
git clone https://github.com/rakibulhasanridoy/keen-keeper.git
```
- 📦 2. Install Dependencies
```js
npm install
```
## Or download the ZIP file and extract it.

### This will install all required packages like React, Tailwind CSS, DaisyUI, and others.

- ▶️ 3. Start the Development Server
```js
npm run dev
```

### If the project uses Create React App, use:

### npm start


- 🌐 4. Open in Browser

### After running the server, open your browser and go to:

http://localhost:5173

or

http://localhost:3000



### Now you can explore and modify the Keen-Keeper locally 🎉
